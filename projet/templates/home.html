À propos
Ces pages web font partie d'un projet scolaire réalisé par un groupe d'étudiants de l'école supérieure de technologie d'agadir, afin de mettre en place une base de données complète et solide pour les projets de location de voitures.

Technologies
FRONT : html5, css3, bootstrap, javascript, jquery, ajax.
BACK : python, flask.
DBMS : mysql.
